# socks5-proxy

RUN
---

```
$ go get -v -u github.com/40t/socks5-proxy
$ cp -rf $(go env GOPATH)/bin/socks5-proxy /usr/local/bin
$ socks5-proxy
```

USAGE
-----

```
$ socks5-proxy -p [port] -a [username:password]
$ socks5-proxy  //default port is 9999
```
